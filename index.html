Description

The rJ Library is a collection of open-source programs made by developers, for developers. It consists of Java utility, including tools and malware in either command-line form, GUI (Graphical User Interface) form, or both.
Created by RSIX as a fun way to see how much can be made with only Java alone.

Fun fact: RSIX stands for Register Six, being a reference to the best zombie game ever: Dead Rising.
