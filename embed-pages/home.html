Description

The rJ Library is a collection of open-source Java projects, consisting of command-line tools and malware. It was created by RSIX as a fun way to see how much can be made with only Java alone.
Fun fact: RSIX stands for Register Six, being a reference to the best zombie game ever: Dead Rising. Now rJ just stands for registerJava with no special reference.
